# ThingLink plugin for Unity

This plugin allows you to quickly export a scene from Unity as a 2D or 360Â° image to ThingLink, a platform for creating interactive multimedia.
You can then quickly share the image with your clients or colleagues via a link, or use ThingLink to add annotations, questions and comments.

# Installation

Editor plugin is already installed and can be found from Unity Toolbar -> Window -> ThingLink
Runtime plugin can be added to your scene by dragging the ThingLinkPluginCanvas from
Assets -> ThingLink -> ThingLinkPlugins -> Prefabs

Learn more at https://www.thinglink.com/

# Detailed instructions & support

https://support.thinglink.com/hc/en-us/articles/10134817735703

# Demo project

Use the demo project included in the package to try out the plugin:

 1. Start the demo
 2. Click the ThingLink icon in the upper-right corner
 3. Log in to your ThingLink account (Don't have an account? Go to https://www.thinglink.com/register)
 4. Move the camera to the desired location and click Upload
 5. Go to https://www.thinglink.com/ to view, edit and share the image on ThingLink
